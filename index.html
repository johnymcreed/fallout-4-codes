Command generator

This generator allows you to generate a custom string for each ID which allows you to not have to constantly type out commands and then copy the code within fallout 4s console.

Commands Usage
%item_id Will be replaced with an items ID for each one.
%item_name Will be replaced with an items NAME for each one.
HTML support You can write html tags for better apperance to you.
